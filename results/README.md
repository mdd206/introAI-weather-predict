# Results

Quy ước lưu kết quả:

- `results/regression/temperature/<model_name>/`: model dự đoán nhiệt độ.
- `results/classification/rain/<model_name>/`: model dự đoán có mưa hay không.
- `results/lstm/lstm_multitask/`: LSTM dự đoán cả nhiệt độ và mưa.

Mỗi model nên lưu:

- file metrics
- file predictions
- file phân tích feature hoặc loss nếu có
