# Regression models

Dùng cho bài toán dự đoán nhiệt độ ngày mai (`target_temp`).

Các model đặt theo từng thư mục thuật toán:

- `lasso_regression/train.py`: đã thực hiện.
- `xgboost_regression/`: sẽ thêm sau.
- `random_forest_regression/`: sẽ thêm sau.

Kết quả lưu vào `results/regression/temperature/`.
